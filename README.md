# AD-DC

This repository contains the cloud-based projects as part of the Application Development and Deployment in Cloud.  
In these projects, I explored how cloud services can be developed and tested locally using Docker, LocalStack, Python, and AWS CLI.

## About the Projects

### localstack-s3-project1
In this project, I worked on the basic setup of LocalStack and understood how to run AWS-like services in a local environment.  
This helped me learn the initial configuration and testing process.

### localstack-s3-project2
In this project, I continued working with LocalStack and focused more on deployment practice and project organization.  
It helped me improve my understanding of how local cloud environments are structured and managed.

### localstack-s3-project3
In this project, I built a serverless application using AWS Lambda and Amazon S3 on LocalStack.  
The Lambda function checks for files in the source bucket, copies them to the destination bucket, and then deletes them from the source bucket.  
This project helped me understand how serverless workflows and event-driven processing work in a practical way.

## Tools and Technologies I Used
- Docker Desktop
- LocalStack
- AWS CLI
- Python
- Boto3
- Visual Studio Code

## What I Learned
Through these projects, I learned how to:
- set up and run LocalStack using Docker
- use AWS CLI commands in a local environment
- create and manage S3 buckets
- write and deploy Lambda functions in Python
- understand the basics of serverless computing and local cloud simulation
