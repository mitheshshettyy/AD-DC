# AD-DC

This repository contains hands-on cloud computing projects developed as part of Application Development and Deployment in Cloud.  
All projects are built and tested locally using tools such as Docker, LocalStack, Python, and AWS CLI.

## Project Overview

### 1. localstack-s3-project1
This project focuses on the basic setup and testing of LocalStack services.  
It is mainly used to understand how AWS-like services can run in a local development environment.

### 2. localstack-s3-project2
This project extends the LocalStack setup with additional deployment practice and configuration.  
It helps in understanding project structure, service setup, and local cloud simulation.

### 3. localstack-s3-project3
This project demonstrates a serverless file transfer workflow using AWS Lambda and Amazon S3 on LocalStack.  
The Lambda function checks a source bucket, copies files to a destination bucket, and deletes them from the source bucket after transfer.

## Tools and Technologies Used
- Docker Desktop
- LocalStack
- AWS CLI
- Python
- Boto3
- Visual Studio Code

## Objective
The main objective of these projects is to learn cloud application development and deployment concepts in a local environment without using actual AWS infrastructure.  
These projects help in understanding services like S3, Lambda, and event-driven execution in a practical way.

## Learning Outcomes
- Understanding LocalStack setup using Docker
- Working with AWS CLI commands locally
- Creating and managing S3 buckets
- Writing and deploying Lambda functions in Python
- Simulating serverless architecture in a local environment

## Conclusion
These projects provide practical exposure to cloud-native development and deployment concepts.  
They serve as a foundation for understanding real AWS services and serverless workflows in a safe and cost-effective local setup.
